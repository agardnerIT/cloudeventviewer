CloudEvents Viewer Help

The CloudEvents viewer exists to help demonstrate the power and value of CloudEvents.

It is a free, open source, in-memory CloudEvent database and UI in a single package.

It is intended to not designed for production usage. Use CloudEvents viewer to prove the value, then build a real production-ready system with other tools.

About CloudEvents

CloudEvents is a standard specification describing events in a vendor neutral way.

CloudEvents can be implemented and trasmitted in any language or transport protocol.

This tool "speaks" CloudEvents in JSON, transmitted via HTTP.

For more information see the CloudEvents website.

Send New CloudEvents

Use CloudEvent Builder

The CloudEvent builder is a user interface to build and send CloudEvents.

If you are new to CloudEvents, it is recommended to start here.

Insert One CloudEvent

curl -X POST http://{{ uri }}:{{ port }}/insert -H "Content-type: application/json" -d '{ CLOUD_EVENT_JSON }'

Bulk Insert Multiple CloudEvents

curl -X POST http://{{ uri }}:{{ port }}/insert -H "Content-type: application/json" -d '[{ CLOUD_EVENT_JSON }, { EVENT_2} ...]'
